##### Title:  **Employee Attrition Prediction for HR Analytics**



##### **## Project Overview**

• Employee attrition is a major challenge for organizations  

• Leads to increased recruitment costs  

• Causes loss of skilled employees  

• Reduces overall productivity  

• Uses Machine Learning techniques for prediction  

• Helps HR departments take proactive decisions  



##### **## Objectives**

• Predict whether an employee is likely to leave  

• Identify key factors influencing attrition  

• Build an interactive dashboard for HR analytics  

• Provide insights for better decision-making  



##### **## Technologies Used**

• Python  

• Pandas  

• NumPy  

• Scikit-learn  

• Streamlit  

• Plotly  



##### **## Dataset**

• Source: Kaggle HR Dataset  

• Contains employee-related features  

• Job Satisfaction  

• Salary  

• Work-Life Balance  

• Years at Company  

• Performance Rating  



##### **## Methodology**

• Data Preprocessing  

&nbsp; • Handling missing values  

&nbsp; • Encoding categorical variables  

&nbsp; • Feature scaling  



• Handling Imbalance  

&nbsp; • Oversampling techniques  



• Model Building  

&nbsp; • Logistic Regression  

&nbsp; • Decision Tree  

&nbsp; • Random Forest  



• Model Evaluation  

&nbsp; • Accuracy  

&nbsp; • Precision  

&nbsp; • Recall  

&nbsp; • F1-Score  



• Explainability  

&nbsp; • SHAP (feature importance analysis)  



##### **## Results**

• Random Forest achieved highest accuracy (~80%)  

• Model handles complex feature relationships  

• Important factors influencing attrition:  

&nbsp; • Job Satisfaction  

&nbsp; • Work-Life Balance  

&nbsp; • Salary  



##### **## Features**

• Upload employee dataset  

• Train machine learning model  

• Predict employee attrition  

• Display prediction results  

• Visualize insights using graphs
